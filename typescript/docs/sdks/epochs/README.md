# Epochs

## Overview

### Available Operations

* [getCurrent](#getcurrent) - Get current epoch
* [list](#list) - List epochs
* [create](#create) - Create epoch
* [get](#get) - Get epoch
* [retry](#retry) - Retry failed epoch
* [verify](#verify) - Verify epoch anchor
* [getTransaction](#gettransaction) - Get epoch anchoring transaction
* [commitRoot](#commitroot) - Commit epoch root
* [commitRootsBatch](#commitrootsbatch) - Commit multiple epoch roots
* [getAnchoringHealth](#getanchoringhealth) - Anchoring service health
* [listReady](#listready) - Get epochs ready for finalization
* [getStats](#getstats) - Epoch statistics
